# Vim Commands that are helpful


## Insert Mode

### Copy Word in insert mode

    <C-x><C-p>

### Copy Line in insert mode

    <C-x><C-l>

### File completion

    <C-x><C-f>


## Normal Mode

**Jump to Definition in a javascript file**

    gd and gD

**Jump Back to previous area
    go

**Specifies Split size

    :10winc "+ - < >"
